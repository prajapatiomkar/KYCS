# Project Setup In Your Local Machine
### Steps:
#### 1) Clone the project
    git clone https://github.com/prajapatiomkar/KYCS.git
#### 2) Move to the project folder
    cd ./KYCS
#### 3) Install the dependency
    npm install 
#### 4) To run frontend 
    cd frontend/ && npm run dev
#### 5) To run backend 
    cd backend/ && npm run server
